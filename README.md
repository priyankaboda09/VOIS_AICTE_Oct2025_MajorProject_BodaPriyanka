# Project Description
This project explores the Netflix catalog dataset containing 7,789 records and 11 columns, including details like title, director, cast, country, release year, rating, duration, type, and genre. 

The aim is to:
 Analyze Movies vs TV Shows over the years
 Identify top genres and their trends
 Compare country-wise contributions to Netflix’s library
 Provide strategic recommendations based on findings

 # Problem Statement
 Netflix faces growing competition from Amazon Prime, Disney+, and regional OTT platforms. Understanding content trends is crucial for:
- Strategic content acquisition and production
- Catering to international audiences
- Identifying underrepresented genres and countries

# Technologies Used
Python – Main programming language for data cleaning, analysis, and visualization.
Google Colab – Interactive environment to write and run code.
Pandas – Load, clean, manipulate, and aggregate dataset.
NumPy – Efficient numerical operations and handling missing values.
Matplotlib – Create static plots like line charts, bar charts, and histograms.
Seaborn – Enhanced static plots with better aesthetics (heatmaps, countplots).
Plotly – Interactive charts (line plots, choropleth maps) for deeper insights.
CSV / Excel – Dataset storage and versioning.
Git & GitHub – Repository hosting, version control, and sharing project code.

# Methodology
1. Load and inspect the dataset
2. Clean data:
   - Handle missing values
   - Normalize column names
   - Parse dates, durations, and types
3. Data Transformation:
   - Explode multi-value columns (genres, countries)
   - Extract release year and month
4. Exploratory Data Analysis (EDA):
   - Movies vs TV Shows trend over years
   - Genre distribution & trends
   - Country-wise contributions
   - Duration analysis (minutes & seasons)
5. Visualization:
   - Static plots (Matplotlib / Seaborn)
   - Interactive charts (Plotly)
6. Strategic insights & recommendations

# Findings 
Netflix’s content library has grown significantly year over year
Movies dominate in certain years, while TV Shows are catching up
Popular genres: Drama, Comedy, Documentary
Underrepresented genres and countries present opportunities for expansion
Recommendations:
Focus on high-demand genres for targeted audiences
Expand content from underrepresented countries
Balance Movies and TV Shows based on audience preferences
